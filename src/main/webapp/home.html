STUDENT RESULT MANAGEMENT SYSTEM

The project titled STUDENT RESULT MANAGEMENT SYSTEM is a web based application developed to maintain the results of the students. This project contains three modules:
1.Home module
2.admin module
3.student module.
Registered users can access the database of the students, which will provide details about subjects, classes, results, and student can download results, which is useful for the students.

REQUIREMENTS ANALYSIS

The proposed system has the following requirements:
1.System has two kinds of users: Student and Teacher.
2.The student can only view results.
3.Teacher must be able to add/ delete/ update/ view student, subject, result information and keep the list up to date.
4.System needs to store information about new entry of Student.
5.System needs to store information about all the subjects and the results of all the students.
6.It needs a security system with a unique username and password given to the admin.
7.The system also requires a search area to search the student result using their USN.
